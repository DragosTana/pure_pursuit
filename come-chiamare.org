 1. compili normale il c++
    #+begin_src bash
      # nella cartella root della repo
      cd pp_cpp
      g++ -o pp pure_pursuit.cpp
    #+end_src
 2. per eseguire il c++
    #+begin_src bash
      # nella cartella root della repo
      ./pp_cpp/pp
    #+end_src
    questo darà l'output del c++
    (da anche testo oltre agli input per lo script, visto che lo standard error (~std::cerr~) non finisce nella pipe lo sto usando per i print di debug)
 3. per eseguire il c++ e darlo al python
    #+begin_src bash
      # nella cartella root della repo
      ./pp_cpp/pp | python pp_python/cpp_visualization.py
    #+end_src
    il testo dello standard error a seconda del terminale potrebbe risultare rosso e/o in grassetto
